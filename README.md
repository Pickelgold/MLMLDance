# MLMLDance

MLMLDance is an audio to 3D animation machine learning pipeline meant to generate a dance from a song. It extracts audio tensors from the audio file's Mel Spectrogram and generates a corresponding set of position tensors, which is converted to an animation.

## Requirements

To run this program, you must have the following installed:

* Bash
* Blender
* Python
* NumPy
* PyTorch
* TorchAudio
* TorchCodec

## Running the Demo

Download the Demo directory and run `./demo.sh`

The full dataset is provided in the Motions directory.
